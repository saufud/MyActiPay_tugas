# MyActiPay

MyActiPay adalah aplikasi **Java Web** untuk pengelolaan kas kegiatan, di mana setiap anggota memiliki target iuran yang dapat dibayarkan secara bertahap (dicicil). Aplikasi ini dikembangkan sebagai **tugas mata kuliah Java Web**.

## Latar Belakang
Pengelolaan kas kegiatan di lingkungan kelas sering kali masih dilakukan secara manual sehingga rawan terjadi kesalahan pencatatan dan kurang efisien. Oleh karena itu, aplikasi MyActiPay dibuat untuk membantu proses pencatatan kas agar lebih terstruktur, transparan, dan mudah dikelola.

## Fitur Utama
- Pengelolaan data anggota
- Pencatatan kas kegiatan
- Penentuan target iuran per anggota
- Pembayaran iuran secara bertahap
- Laporan data kas

## Teknologi yang Digunakan
- Java Web (JSP & Servlet)
- Apache Tomcat
- MySQL
- NetBeans IDE

## Struktur Project
Project ini menggunakan struktur standar **Java Web NetBeans (Ant)**:
- `src/` : Source code Java
- `web/` : File JSP, HTML, dan resource web
- `nbproject/` : Konfigurasi NetBeans
- `build.xml` : Konfigurasi build project
